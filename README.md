# Inovatech 2026

Frontend em React + Vite para a plataforma de vigilância do mercúrio no Rio Negro.

## Visão geral

O projeto apresenta uma interface de monitoramento para apoiar leitura territorial, priorização de comunidades e comunicação de risco ambiental e de saúde pública.

Nesta versão, o foco está no frontend:

- hero com posicionamento do problema
- painel inicial de risco
- arquitetura do sistema
- leitura territorial do eixo do Rio Negro
- comunidades monitoradas
- base de evidências
- estrutura pronta para integração futura com backend e banco

## Stack

- React 19
- TypeScript
- Vite
- Lucide React

## Como rodar

```bash
npm install
npm run dev
```
